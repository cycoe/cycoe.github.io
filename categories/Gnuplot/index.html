Gnuplot 绘图第二弹-标签与图例

上一弹中介绍了 Gnuplot 能够绘制的 2D 图形，这一弹接着来介绍一下 Gnuplot 中的标签 与图例。

设置标签 

 1 set xrange [0:4*pi]
 2 set yrange [-1.2:2]
 3 set y2range [0:2]
 4 set ytics nomirror
 5 set y2tics 0, .1
 6 set style fill pattern 5
 7 set xlabel "Time (sec.)"
 8 set ylabel "Amplitude of Waves"
 9 set y2label "Difference"
10 # 文本需要手动换行
11 set title "Amplitude with time\nAnd Difference between waves"
12 
13 # 绘制每一条线的 title 用来设定图例文本
14 plot '+' using 1:(sin($1)):(cos($1)) with filledcurves notitle,\
15      sin(x) w l lw 2,\
16      cos(x) w l lw 2,\
17      '' using 1:(abs(sin($1)-cos($1))) w l lw 2 axis x1y2 title "Difference"

改变标签字体大小 

 1 # 全局字体
 2 set term png font 'Arial, 14'
 3 
 4 # 单独设置某一部分的字体大小
 5 set xlabel font ",18"
 6 set ylabel font ",18"
 7 set y2label font ",18"
 8 # 同时设置内容和大小
 9 set title "Amplitude with time\nAnd Difference between waves" font ",24"
10 
11 replot

设置图例的位置和字体等属性 

1   # spacing 调整图例之间的间距，默认值是 1.25
2   set key at graph .4, .95 spacing 1
3 
4 replot

图例显示在图外面 

1 set term png size 500,200
2 set key outside
3 
4 plot sin(x) title "sine", cos(x) title "cosine"


1 set key box lt -1 lw 2
2 
3 replot

带箭头的文本 

1 # left at 后面的数字是 x1y1 坐标轴，offset 后面是以字体网格为坐标
2 set label "Max Difference" left at 8, 1.5 offset .5, 0
3 # arrow 后面的 1 是一个 tag，front 是告诉 gnuplot 在最前面绘制 arrow
4 # size 中的第一个数字是箭头倒钩的长度，第二个数字是倒钩的角度
5 set arrow 1 from first 8, 1.5 to pi*7/4, 1 lt -1 lw 2 front size .3, 15
6 
7 replot

在标签和标题中使用 Unicod 字符，需要注意三点：

  • 并不是所有的 Term 都支持 Unicod 字符，比如 PostScript 不支持，pdf 和 png 后端 支持；
  • 设置编码方式为 utf8 ，这一步并不是必需的，如果系统本身编码已经是 utf8 则不用重 新设置；
  • 设置支持 Unicod 的字体。
1 set term png
2 set encoding utf8
3 set title "振幅随时间的变化\n差值随时间的变化" font "Sarasa Term SC,18"
4 
5 replot

Gnuplot 绘图第一弹-二维数据绘图

Gnuplot is a portable command-line driven graphing utility for Linux, OS/2, MS Windows, OSX, VMS, and many other platforms. The source code is copyrighted but freely distributed (i.e., you don't have to pay for it). It was originally created to allow scientists and students to visualize mathematical functions and data interactively, but has grown to support many non-interactive uses such as web scripting. It is also used as a plotting engine by third-party applications like Octave. Gnuplot has been supported and under active development since 1986.

我等科研狗选择的图形界面绘图工具一般是 Origin，无奈 Origin 并不支持 Linux，另一 方面希望选择一款命令行下的绘图工具可以更好地整合至我的工作流中。Linux 中可供选择 的主要有 Gnuplot 和 Matplotlib，Matplotlib 是 Python 的一个第三方模块，可生成印 制质量的数据图。但是当我看到 Gnuplot 的官方 Demo 时，一下就让我心动了，Gnuplot 的默认配色也太好看了，清新淡雅。
