Linux下常用软件和配置记录

发表于 2018-11-09 | 分类于 Linux |

随着使用 Linux 系统的年头越来越长，积累的相关技巧也越来越多。最近一次重装系统突然发现很多配置都已经忘的差不多了，想要配置系统还需要去网上到处搜刮资料实在是麻烦，因此打算用一篇博客记录关于 Linux 系统的配置及常用软件的介绍。

Archlinux 配置

让 Qt5 程序使用原生 GTK+ 主题

Qt5 程序在 GTK3 环境下主题不统一真的很难受，尤其是 GTK3 用了黑色主题而 Qt5 还是刺眼的白色主题的时候。不像在 KDE 设置中设置 GTK3 主题那么方便，设置 Qt5 主题可需要好好折腾一番。而且这个解决方案也是我使用了近一年后才找到的答案。

注销后重新登录，此时打开 qt5ct 应该就不报错了。但此时你会发现主题列表中只有 Qt5 自带的几个样式，而没有 Gtk3 主题。你需要安装 qt5-styleplugins 来使 Qt5 支持 GTK2 主题。你没有看错，我也不知道为啥 Qt5 只支持 GTK2 的主题。但是我们要如何设置 GTK2 主题呢，好吧，又需要安装一个叫 gtk-theme-switch2 的工具。在 gtk-theme-switch2 中设置好满意的 GTK2 主题后，再在 qt5ct 中设置 Qt5 主题为 GTK2 即可。

常用软件

用了几年 Linux 后发现其实 Linux 平台的软件并不少（当然 Adobe 全家桶和 MS Office 除外，要做设计或者办公还是老老实实 Windows 吧:speak_no_evil:），只是与 Windows 和 Mac 平台常用的软件不同而已。Linux 的真正强大之处是命令行软件，几条简单的命令组合就可以完成复杂的功能。

Dia

一个非常小的流程图绘制软件，但是功能够用。

ncdu

ncdu (NCurses Disk Usage) 是一个基于 NCurses 系统的磁盘用量分析命令行软件，相似功能的还有基于 GTK+ 的 Baobab（从版本号来看应该是属于 Gnome 桌面套装）。其实对我来说真的很少使用此类软件，毕竟我很少删除磁盘文件，但是最近 /Home 分区接近爆满让我很难受，还是需要从磁盘的角落翻出以后再也用不上的大文件以删除之。

xreader

pdf 文档阅读器，X-Apps 计划中的一员（这个计划好像只有个阅读器？:laughing:）。体积小巧，优点是可以使用 hjkl 等 vim 键位进行控制。缺点是没有高级功能，单纯作为一个阅读器还是不错的。

pcmanfm

LXDE 桌面环境默认文件管理器，实话说比 Gnome 的鹦鹉螺好用太多，功能上与 Dolphin 相近。Archlinux 上有单独提供 pcmanfm-gtk3 的安装包，其他发型版上应该默认使用的就是 GTK3 了。

lxappearance

GTK+ 界面主题设置工具，又是 LXDE 的一员。LXDE 自带的几款软件质量都挺高，界面风格统一并且功能强大。

gpickview

LXDE 默认的图片查看器，体积小巧，没有太多花哨功能。
