Gnuplot 绘图第四弹-调整刻度

现在，我们已经能够使用 Gnuplot 绘制出自己想要的图形，接下来在第四弹中，我们将对 刻度进行更细致的调整。

启用副刻度

1 set sample 500
2 set grid
3 # x 轴每个主刻度拥有 4 个副刻度
4 set mxtics 4
5 # y 轴每个主刻度拥有 2 个副刻度
6 set mytics 2
7 
8 plot [0:2*pi] sin(12*x)*exp(-x/4)

第二坐标轴上的刻度

1 set ytics nomirror
2 # y2 主刻度的间距是 0.4
3 set y2tics 0.4
4 set my2tics 4
5 set xtics pi/4.
6 set mxtics 4
7 set grid
8 
9 plot [0:2*pi] sin(x) axis x1y1, 2*cos(8*x)*exp(-x) axis x1y2

调整刻度的长度

1 # 将主刻度的长度调整至默认值的 3 倍。如果没有特殊指定，副刻度的长度是主刻度的一
2 # 半，可通过 =set tics scale a, b 同时调整主副刻度的长度
3 set tics scale 3
4 set mxtics 4
5 set mytics 4
6 
7 plot [0:4*pi] sin(x)/x notitle

移除所有的刻度

1 # Gnuplot 中有两种表达方式，=unset tics= 和 =set notics= 等价
2 # 同样地，=unset border= 也和 =set noborder= 等价
3 unset tics
4 plot [0:2*pi] x**2*sin(x)

精细调整刻度值

 1 set size ratio 1
 2 # 设置极坐标
 3 set polar
 4 # 按坐标系的位置设置坐标轴，并关闭副轴
 5 set xtics axis nomirror
 6 set ytics axis nomirror
 7 set zeroaxis
 8 unset border
 9 set border polar
10 unset key
11 set samples 500
12 set grid
13 set rrange [0:11]
14 # 关闭笛卡尔坐标系下 x, y 轴刻度
15 unset xtics
16 unset ytics
17 # 设置极坐标系下的 r 和 t 轴，其中三个数字分别表示起始，间隔和终点
18 set rtics 0,2,10
19 set ttics 30
20 
21 plot [0:2*pi] 4*(1-sin(t))

刻度朝外

1 set tics out
2 set grid
3 
4 plot [-3:3] tanh(x)

手动设置刻度标签

1 set xtics ("{/Symbol p}" pi, "{/Symbol p}/2" pi/2,\
2     "2{/Symbol p}" 2*pi, "3{/Symbol p}/2" 3*pi/2, "0" 0)
3 # add 关键字用于添加标签
4 set xtics add ("3{/Symbol p}/4" pi*3/4)
5 
6 plot [0:2*pi] sin(x)

时间和日期标签

1/1/2011 72.1
1/3/2011 12.2
1/9/2011 9.8
1/1/2012 73.5
1/3/2012 63
1/7/2012 58.9 
 1 # 告诉 Gnuplot x 轴的数据是时间
 2 set xdata time
 3 # 又什么格式解析数据中的时间
 4 set timefmt "%d/%m/%Y %H:%M"
 5 # x 轴标签的输出格式
 6 set format x "%d/%m/%y"
 7 # 旋转 xtics -30 度
 8 set xtics rotate by -30
 9 set rmargin 7
10 
11 plot data using 1:2 with linespoints notitle

在时间标签中启用地域设置

 1 # 告诉 Gnuplot x 轴的数据是时间
 2 set xdata time
 3 # 启用地域设置
 4 set locale "zh_CN.utf8"
 5 # 又什么格式解析数据中的时间
 6 set timefmt "%d/%m/%Y %H:%M"
 7 # x 轴标签的输出格式
 8 set format x "%Y年%b"
 9 set xtics rotate by -30
10 set rmargin 8
11 
12 plot data using 1:2 with linespoints notitle

设置不同地域的数字格式

此处本来应该用 set decimalsign locale "es_ES.utf8" 来做演示才能看出区别，但是不 知为什么我的电脑上好像只有 "en_US.utf8" 和 "zh_CN.utf8" 可用，可能是我在系统中只 开启了这两个编码。 

1 set decimalsign locale "zh_CN.utf8"
2 # 手动设置数字分隔符
3 set decimalsign "|"
4 plot [0:1] x**3 notitle

格式化刻度标签

1 set ylabel "Output power"
2 set format y "P = %.2f Watts"
3 set format x "%.3f%%"
4 # Axes 的右侧与 Canvas 的右侧间距 6 个字符宽度，防止最后一个标签跑到外面去
5 set rmargin 6
6 
7 plot [0:100] x**2 notitle

这里有个小技巧，我们可以通过 format 的方式隐藏所有的标签 

1 set format y ""
2 set format x ""
3 unset key
4 set grid
5 
6 plot x, -x
