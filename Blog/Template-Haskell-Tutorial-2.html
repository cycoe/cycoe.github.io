1. 引用

如你所见，TH 能够生成和处理的语法树一点也不简单。同样不幸的是，可能会生成压根没法编译的 Haskell 代码。换言之，手写语法树既冗长又易出错。

幸运的是，有一种叫做引用的方法可以从任意 Haskell 代码中获取语法树。通过启用 TemplateHaskell 扩展我们将获得 4 种类型的引用：

生成的东西 引用语法 类型
声明 [\textbar{}d ... \textbar{}] Q [Dec]
表达式 [\textbar{}e ... \textbar{}] Q Exp
带类型的表达式 [\textbar{}\textbar ... \textbar{}\textbar{}] Q (TExp a)
类型 [\textbar{}t ... \textbar{}] Q Type
模式 [\textbar{}p ... \textbar{}] Q Pat
